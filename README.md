# Forest Shuffle — Solo Automa (PWA + Offline)

## Como usar (recomendado)
Por segurança do navegador, o Service Worker só funciona via **HTTPS** ou em **localhost**.
Então, para usar offline, rode um servidor local simples:

### Opção 1 (Python)
```bash
cd forest_shuffle_automa_pwa
python -m http.server 8000
```
Abra: http://localhost:8000

### Opção 2 (Node)
```bash
npx serve .
```

## Instalar no celular/PC
Depois de abrir no navegador, clique em **Instalar app** (quando aparecer).
Após instalar, o app funciona **offline** (cache).
